## 🚀 How to Run

### 1. Backend
```bash
cd projectbackend
mvnw clean install
mvnw spring-boot:run
```
**Runs on:** http://localhost:8080

### 2. Frontend
```bash
cd project
npm install
npm run dev
```
**Runs on:** http://localhost:5173

### 3. No Agora Server Needed
Agora is a **cloud service**. Your app connects directly to Agora's infrastructure using:
- App ID (public)
- Token (generated by backend)

## 🔐 Security Architecture

```
Frontend (React)
    ↓ (requests token with JWT)
Backend (Spring Boot)
    ↓ (generates token with App Certificate)
Agora Cloud Service
    ↓ (validates token)
Video/Audio Streams
```

### Security Layers:
1. **JWT Authentication** - User must be logged in
2. **Backend Token Generation** - App Certificate never exposed
3. **Agora Token Validation** - Agora validates token server-side
4. **Time-Limited Tokens** - 1-hour expiration
5. **Role-Based Access** - Doctor (Publisher) vs Patient (Subscriber)

---

## 📡 API Flow

### Token Request Flow:
```
1. User clicks "Start Call"
2. Frontend calls: GET /api/agora/token
   - Headers: Authorization: Bearer <JWT>
   - Params: channelName, userId, role
3. Backend validates JWT
4. Backend generates Agora token
5. Backend returns: { token, channelName, userId }
6. Frontend joins Agora channel with token
7. Audio/Video streams established
```

## 🔍 Troubleshooting

### Backend Issues:
- **Token generation fails**: Check Agora credentials in application.properties
- **401 Unauthorized**: Ensure JWT is valid and included in request
- **CORS errors**: Verify CORS configuration in AgoraController

### Frontend Issues:
- **Cannot access camera/mic**: Check browser permissions
- **Token fetch fails**: Verify backend is running and JWT is valid
- **No remote video**: Ensure both users joined same channel
- **Audio not working**: Check browser audio permissions

### Agora Issues:
- **Invalid App ID**: Verify APP_ID in VideoCall.tsx
- **Token expired**: Tokens expire after 1 hour, rejoin to get new token
- **Connection failed**: Check internet connection and firewall
